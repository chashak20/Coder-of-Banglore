# Coder-of-Banglore 🏙️

## Overview
A Python-based data pipeline to parse and analyze 
Instagram follower profiles of OpenAI — completed 
as a 24-hour challenge.

## Problem Statement
Analyze raw Instagram follower data to answer:
- 👤 Who has the maximum posts?
- 📈 Who has the maximum followers?
- 🔍 Who follows the most people?
- 🗂️ How many unique profile categories exist?
